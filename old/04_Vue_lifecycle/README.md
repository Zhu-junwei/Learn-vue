# Vue生命周期

生命周期的八个阶段:每个阶段会自动执行一个生命周期方法(钩子),让开发者有机会在特定的阶段执行自己的代码

| 状态            | 阶段周期  |
|---------------|-------|
| beforeCreate  | 创建前   |
| created       | 创建后   |
| beforeMount   | 挂载前   |
| mounted       | 挂载完成  |
| beforeUpdate  | 数据更新前 |
| updated       | 数据更新后 |
| beforeUnmount | 组件销毁前 |
| unmounted     | 组件销毁后 |
